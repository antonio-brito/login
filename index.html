Global MicroApps Login

Global MicroApps Login

Introduction

This api implements a client to connect with a service that you can use to

  • Authenticate and get an access token
  • Get the user info of a jwt token
  • Get an access token
  • Change the password if you are authenticated
  • Ask for a code if you forgot your password that your going to receive in your email
  • Confirm your code an set your new password
  • Create a new user to be used in the application

Getting Started

To use the Login API we need that the follow apis were initializad

  • GlobalKit_MicroApps_AppManagement
  • GlobalKit_MicroApps_Bus

After that we need to initialize the service in the bus with this method. LoginFactory.initLoginServiceFactory()

Examples

Login

/* If you want to authenticate you can send a message to the bus like this */

Bus.publishAsync("login", RequestMessage(parameters = ""{"username":"juanito.perez@dedge.global","password":"ValidPassword"}""",id = "1"))!!
    .subscribe(
        AllObserver(
            {
                println("onNext...........")
                    when(it.isSuccess){
                        true -> {
                            val token: TokenDataChild = Json.parse(TokenDataChild.serializer(), it.result!!)
                            println(token.accessToken)
                        }
                        false ->{
                            println("${it.error.toString()}")
                        }
                    }
            },
            {println("onError...........")},
            {println("onComplete...........")}
        )
    )

    // This is just to show the result
    Thread.sleep(3000L)
